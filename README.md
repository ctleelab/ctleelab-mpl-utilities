# ctleelab Matplotlib Plotting Utilities
Helpful plotting utilities for getting publication quality figures which are compatible with touch-up workflows using vector graphics software.

## Installation

You can install using the beta `pixi-build` feature.

```pixi.toml
[workspace]
...
preview = ["pixi-build"] # Activate beta pixi-build feature
...

[dependencies]
ctleelab_plothelper = { git = "https://github.com/ctleelab/ctleelab-mpl-utilities", branch = "main" }
```

Alternatively, the repository can be installed as a local PyPI package.
